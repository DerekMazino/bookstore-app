# Libraries Sofware 3
## Instrucciones de instalacion

ejecutar el siguiente comando en una terminal
```
npm install 
```
en cada una de las siguientes carpetas:  
>frontend-library 

>frontend-review 

>review nodejs

y dejar que el proceso termine
